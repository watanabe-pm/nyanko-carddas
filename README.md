# 🐾 ニャンコカードダスバトル

FC「SDガンダム外伝 ナイトガンダム物語３ 伝説の騎士団」（1992）の
カードダスバトルモードを参考にした、猫テーマのカードバトルゲームです。

## 遊び方

1. 手札からカードを1枚クリックして使用
2. MPが足りない場合は「スキップ」
3. 相手のHPを0にすれば勝利

## カードステータス

| 項目 | 説明 |
|------|------|
| ATK | 攻撃力（相手DEFを引いた値がダメージ） |
| DEF | 防御力（相手ATKから引かれる） |
| MP  | 使用コスト（毎ターン3回復、最大10） |

## 特殊効果

| 効果 | 内容 |
|------|------|
| ひなたぼっこ | HP +10 回復 |
| 好奇心       | 2枚追加ドロー |
| 爪とぎ       | 相手のDEFを無視して攻撃 |
| ねこパンチ連打 | 2回攻撃 |
| 毛玉         | 相手に毒（3ターン×5ダメージ） |
| まるまり     | このターンの被ダメージを50%軽減 |

## レアリティ

- **N**（白枠）：ノーマル
- **R**（青枠）：レア
- **SR**（金枠）：スーパーレア
