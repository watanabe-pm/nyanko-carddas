# CLAUDE.md

## プロジェクト概要

猫カードで人間キャラクターを「倒す」か「魅了する」2ルートのカードバトルゲーム。
フロントエンド完結（HTML / CSS / JavaScript）。GitHub Pages で公開。

## 仕様書

`spec-v0.4.md` を必ず参照すること。実装前に該当箇所を確認する。

## ファイル構成

```
nyanko-carddas/
├── index.html
├── css/
│   └── style.css
├── js/
│   ├── cards.js      # カードデータ定義
│   ├── items.js      # アイテムデータ定義
│   ├── enemies.js    # 人間キャラクターデータ定義
│   ├── game.js       # ゲームロジック
│   └── ui.js         # UI描画・イベント処理
├── spec-v0.4.md      # 仕様書
├── CLAUDE.md         # このファイル
└── README.md
```

## 実装ルール

1. **実装前に設計方針を説明する**：コードを書く前に、何をどう実装するか説明してから着手する
2. **1ファイルずつ実装する**：複数ファイルを一度に変更しない。1ファイル実装→確認→次へ
3. **コメントは日本語で書く**：コード内のコメントはすべて日本語
4. **仕様書に忠実に実装する**：仕様にない機能を勝手に追加しない
5. **不明点は実装前に確認する**：仕様が曖昧な場合は実装せずに質問する
6. **既存ファイルを変更する前に内容を確認する**：上書きミスを防ぐ

## 技術スタック

- HTML / CSS / JavaScript（Vanilla）
- 外部ライブラリなし
- localStorage 使用可（デッキ・進行状況の保存）

## 注意事項

- バックエンドなし・サーバーサイド処理なし
- GitHub Pages での公開を前提とする
- スマホ対応（レスポンシブ）を考慮する
